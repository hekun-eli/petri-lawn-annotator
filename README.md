# Petri Lawn Annotator

批量识别培养皿中的菌苔扩散范围和中心纸片，自动生成绿色 + 红色标注。

## 功能

1. **菌苔标注**（🟢 绿色）— 识别细菌扩散范围并描边
2. **纸片标注**（🔴 红色）— 检测中心白色圆形纸片并描边
3. **批量处理** — 自动跳过失败图片

## 工作流程

```
输入图片（灰度，黑底）
  │
  ├─ Step 1: 菌苔绿色标注
  │   ├─ 蒙版非黑区域（培养皿区域）
  │   ├─ OTSU 自适应阈值 + 可调偏移量
  │   ├─ 形态学操作（闭运算 + 开运算）
  │   ├─ 取含图像中心的最大连通域
  │   ├─ 实心蒙版 → Canny 边界提取
  │   └─ 膨胀加粗 → 绿色叠加
  │
  ├─ Step 2: 纸片红色标注
  │   ├─ 中心区域 HoughCircles 圆形检测
  │   ├─ 内外环亮度差验证
  │   └─ 绘制红色圆环
  │
  └─ 输出合并结果 PNG
```

## 检测方法

### 菌苔检测

基于 OTSU 自适应阈值的连通域分析：

1. 排除黑背景，只分析培养皿内部
2. `cv2.threshold(..., THRESH_BINARY + THRESH_OTSU)` 自动计算最佳阈值
3. 叠加 `THRESHOLD_OFFSET` 上调（值越大标注越保守）
4. 形态学闭运算 + 开运算去除噪点
5. 取包含图像中心的最大连通域作为菌苔
6. Canny 边缘提取 → 膨胀加粗 → 绿色叠加

### 纸片检测

基于 HoughCircles 的圆形检测：

1. 在图像中心 240×240px 区域内搜索
2. 多组参数尝试 `dp ∈ [1.0, 1.2, 1.5]`，`param2 ∈ [15, 20, 25, 30]`
3. 半径范围：`PAPER_MIN_RADIUS` ~ `PAPER_MAX_RADIUS`（默认 10~80px）
4. 验证：圆内平均亮度与周围环带差异 > 15 即确认为纸片
5. 绘制红色圆环

## 参数说明

### 菌苔参数（在 `petri_annotator.py` 顶部修改）

| 参数 | 默认值 | 说明 |
|---|---|---|
| `LAWN_THRESHOLD_OFFSET` | 10 | OTSU 阈值上调量（越大越保守，推荐 10~20） |
| `LAWN_LINE_WIDTH` | 4 | 绿色边界线宽（px） |

### 纸片参数

| 参数 | 默认值 | 说明 |
|---|---|---|
| `PAPER_MIN_RADIUS` | 10 | 纸片最小半径（px） |
| `PAPER_MAX_RADIUS` | 80 | 纸片最大半径（px） |
| `PAPER_LINE_WIDTH` | 3 | 红色边界线宽（px） |

## 使用方法

### 完整流水线（菌苔 + 纸片）

```bash
python3 petri_annotator.py [输入目录] [输出目录] [OTSU偏移量]
```

默认运行：

```bash
python3 petri_annotator.py
```

自定义路径和参数：

```bash
python3 petri_annotator.py /path/to/input /path/to/output 15
```

### 仅菌苔标注

```bash
python3 lawn_only.py [输入目录] [输出目录]
```

## 输入要求

- 灰度 PNG 图片（或彩色图，会自动转灰度）
- 背景为黑色（培养皿以外区域已置黑）
- 培养皿中心大致在图像中心
- 纸片为圆形，位于培养皿中心附近

## 输出

| 内容 | 颜色 | 说明 |
|---|---|---|
| 菌苔边界 | 🟢 `(0, 255, 0)` | 沿菌苔实际边缘走线，保留凹陷 |
| 纸片轮廓 | 🔴 `(0, 0, 255)` | HoughCircles 精确定位圆 |
| 最远距离 | 🟡 `(0, 255, 255)` | 纸片圆心→菌苔最远边界（不跨越绿线）|

## 输出文件结构

```
输出目录/
├── DSCFxxxx.png      ← 标注后图片
├── DSCFxxxx.png
└── ...
```

## 距离测量（measure.py）

基于 `result_combined` 的标注结果（红圆 + 绿边），测量扩散距离。

**测量规则：**
- 红色纸片中心 → 绿色菌苔边界上最远点
- 黄色连线 = 全程在绿色区域内部，不跨越任何绿线
- 红色圆直径 = 8mm（标尺），换算毫米值
- 图片上无文字标注

**输出 Excel 字段：**

| 字段 | 说明 |
|---|---|
| 文件名 | 图片名称 |
| 纸片圆心X/Y | 检测的纸片中心（px） |
| 纸片半径/直径 | px 值 |
| 比例(px/mm) | 纸片直径 8mm 标尺换算 |
| 最远点X/Y | 菌苔边界上最远点（px） |
| 最远距离(px/mm) | 扩散距离 |
| 最远角度(°) | 最远点方位 |
| 菌苔面积 | px² 和 mm² |

```bash
python3 measure.py [输入目录] [输出目录]
# 默认: result_combined/ → result_measure/
```

**验证机制：**
- Flood fill 法生成绿色内部区域
- 线段 100 段逐点采样
- 排除端点附近 4% 像素（绿线厚度）
- 确认每一点都不在绿色边界上
- 提前 15px 终止绘制，视觉上零重叠
